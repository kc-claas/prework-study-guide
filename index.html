Git

  • git status: checks what branch we are currently on
  • git checkout -b insert-branch-name-here: creates a new branch and switches to it
  • git add -A: stages all files in the working branch
  • git commit -m "insert-message-here": commits changes, and adds a descriptive message
  • git pull origin main: checks to sync up local branch with main remote branch
  • git push origin insert-branch-name-here: pushes changes to remote branch
